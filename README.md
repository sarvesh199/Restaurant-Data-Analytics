# Restaurant Data Analytics

This repository contains data, code, and analysis for exploring restaurant-related datasets (sales menus). The project focuses on cleaning, exploring, modeling, and visualizing restaurant data to answer business questions such as revenue drivers, customer sentiment, and location-based performance.

**Primary Goals**

- **Data cleaning:** ingest raw CSV/JSON, handle missing values, normalize fields.
- **Exploration & visualization:** summary statistics, time series of sales, heatmaps of locations, review sentiment.
- **Modeling & insights:** identify features correlated with revenue or ratings; basic predictive models and segmentation.
- **Reproducibility:** provide notebooks and scripts to reproduce analysis and figures.

**Repository Layout (recommended)**

- **data/**: raw data to process.
- **RestaurantDataAnalysis/**: Schema Validation And Business Transformation
- **DataVisualization/**: Databricks dashboard to answer the business questions
- **README.md**: this file.
